- Ai systems = Code + Data
- 以数据为中心，而不是以算法为中心
	- 通常情况下，提升数据要比提升算法的收益大很多。甚至于提升算法的收益为零
- 机器学习项目生命周期
	- 定义项目
	- 收集数据
	- 训练模型
	- 部署生产
- 当项目进行到部署生产时，整个工作才算完成一半，接下来需要监控输出质量，并周期性回溯数据，并从一开始的步骤开始，迭代提升整体性能
- 数据标注的一致性很重要
	- 数据标注的方式有很多，一致性很重要。当有多种标注方式时，因全部选择同一种标注方式
	- 对于多个标注者来说，交叉验证有必要
- 数据质量要比数据数量更重要
	- 提高数据质量通常能比扩充数据数量更能提升模型性能
- MLOPs 的主要任务是在整个机器学习项目生命周期中保持高质量的数据
